About the Loot Generator

The Fantasy Loot Generator is designed to facilitate generation of random loot using the tables presented in the Savage Worlds Fantasy Companion. Below are a few notes regarding the decisions governing the script and features.

Multiple Rolls

Users can now roll for multiples of a given treasure rating. For example, a user can roll Meager treasure four times all at once. The output will include the gold and any potential relics for each treasure as well as a total amount of gold (not including the cost of any relics).

Links to Individual Items

I've added the ability to link to or bookmark individual items generated via a query string. This is a great way to send individual items to others, bookmark them yourself, or use them to send a bug report to me with an example.

Print Multiple Cards

Since you'll be able to generate multiple items, you should be able to print multiple cards to a single page. Using the browser's print feature, you'll be able to print approximately 2.5" x 3.5" cards for items generated. The CSS is still being tweaked, but it's my goal to have at least 4 cards per sheet if not 6 cards per sheet.

Google Cloud Print

I've added URLs and print icons for generated relics to print using Google Cloud Print. This allows users to print to any printer they own and have set up in Google Cloud Print, send the item's URL to another device that has Google Chrome, or save a PDF to Google Drive. The output will always be in the print card layout (still undergoing some tweaks) regardless of whether it's printed as a page or saved as a PDF. If you're using Google Chrome, Cloud Print is baked in to the default printing command, and you'll be able to save or print entire collections of items generated in one document.

Minor and Major Artifacts

Fixed Aspects: For Powers of Minor or Major Artifacts, the Fantasy Companion instructs, "Powers which can affect multiple aspects (such as boost trait) must have a single, fixed aspect chosen during creation." Accordingly, the script randomly generates the fixed aspect such as a Trait to which the power applies and sometimes an Attribute associated with the type of armor or weapon as suggested by the table descriptions. For example, boost Smarts or boost Spirit applies to a helmet, and boost Strength or boost Vigor applies to Melee Weapons.

Power Points for Major Artifacts: The script randomly rolls for a specific number of Power Points or an effect (typically determined by how many Power Points are spent to activate the power). If an effect is given, then it is fixed. For example, an item with the blast power might always produce a Medium Burst for 3d6 damage.

Durations: The script does not determine the duration of an item's power. It always assumes the minimum duration. GM's are free to extend the duration by adding the appropriate Power Points and adjusting the cost accordingly.

Named Items

The Ranged Weapons and Miscellaneous Magic tables don't include results for named items unlike the Armor & Shields and Melee Weapons tables. According to Savage Worlds Core Rules Brand Manager Clint Black, "If you want to add a random chance, then for Ranged Weapons, got [sic] to Table 3G if you roll a second 20 on Table 3B. And for Miscellaneous Magic, go to Table 4F if you roll a second 20 in Table 4A." The script follows this method exactly. However, there is an option to adjust the chance of named ranged weapons and miscellaneous items to 5%. A similar option to adjust Melee Weapons to 5% from the reprints change to 2.5% is being develped.

Power Points for Scrolls

Power Points for Scrolls are based on the defaults listed in Table 7B: Powers.

Intelligent Relics

Attributes: The script rolls d2 a number of times equal to the result from the Attributes column. Depending on whether the result is a 1 or 2, it increases die size for Smarts or Spirit respectively.

Potions, Scrolls, and Tomes: If the item is a potion, scroll, or tome, then the script does not roll for Intelligent Relics.
